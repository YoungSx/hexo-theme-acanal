# Acanal

## Install

``` bash
$ git clone https://github.com/YoungSx/hexo-theme-acanal.git themes/acanal
```

## Enable

Modify `theme` setting in `_config.yml` to `acanal`

``` yml
theme: acanal
```

## Update

``` bash
cd themes/acanal
git pull
```
